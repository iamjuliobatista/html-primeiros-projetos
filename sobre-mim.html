Sobre Mim

Formado em Educação Física pela Universidade Nove de Julho, passo por uma transição de carreira. Sempre fui apaixonado por tecnologia, curso o último semestre de Análise e Desenvolvimento de Sistemas pela Universidade Nove de Julho e em meus estudos conheci a área de Cloud Computing (possuo a certificação AWS Cloud Practitioner), possuo conhecimento na criacao de páginas web (HTML, CSS e JS) já estudei as linguagens Java, Kotlin e trabalhei com as IDE 's VSCode, Eclipse, IntelliJ e Android Studio.

Em bootcamps que participei pude desenvolver páginas web desde o zero, aplicações Java simples que rodava no próprio console da IDE e até projetos mais complexos que envolviam criação e consumo de API Rest, criação de frontend em Kotlin em uma estrutura de aplicação KMM. Busco oportunidade para desenvolver habilidades nas áreas de Cloud Computing e Programação, como AWS, programação back-end e front-end, entre outras áreas. Meu objetivo é ingressar como cargo Jr e me desenvolver e crescer junto com a organização.
